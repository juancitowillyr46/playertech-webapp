# FE-011 Dashboard Operativo

## Objetivo

Ofrecer una visión resumida del estado operativo de la academia.

## Problema que Resuelve

El usuario necesita una vista rápida del negocio sin navegar por todos los módulos.

## Valor de Negocio

Mejora la toma de decisiones y la priorización operativa.

## Actores

* Tenant Owner
* Academy Admin
* Staff autorizado

## Dominios Involucrados

* Player
* Membership
* Payments
* Academy

## Historias de Usuario

* HU-001 Ver jugadores activos.
* HU-002 Ver pagos pendientes.
* HU-003 Ver matrículas activas.
* HU-004 Ver resumen de caja o cartera.

## Reglas de UX Relacionadas

* Priorizar métricas de uso frecuente.
* Presentar tarjetas o bloques de lectura rápida.
* Evitar un dashboard decorativo sin utilidad operativa.
