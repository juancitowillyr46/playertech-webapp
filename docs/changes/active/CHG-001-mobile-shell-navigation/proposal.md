# CHG-001 Mobile Shell Navigation

## Resumen

Ajuste del menú lateral en mobile para mantener legibilidad, preservar labels y evitar una presentación reducida a solo iconos.

## Motivo

La navegación actual en mobile se ve desproporcionada y pierde contexto visual. Esto afecta usabilidad y comprensión de rutas.

## Alcance

Incluye:

* sidebar mobile;
* burger del topbar;
* overlay y cierre;
* visibilidad de textos;
* ancho del panel;
* bloqueo de scroll.

Excluye:

* cambio de navegación de negocio;
* pruebas automatizadas en esta fase;
* refactor de features funcionales.

## Objetivo de negocio

Mejorar la navegación primaria en mobile para que el shell siga siendo claro y usable.

## Estado

Active.
